# 配置文件：
- Pixels : 从bing下载壁纸的分辨率。可选 
	; 1920x1200   1920x1080    1366x768   1280x768    
	; 1024x768    800x600       800x480   768x1280  
	; 720x1280    640x480       480x800   400x240     
	; 320x240     240x320 
- lockimg：是否收集win10锁屏壁纸
- bingimg: 是否从bing下载今日壁纸
- autosetbingimg：是否自动将bing今日壁纸设置为桌面壁纸，必须设置bingimg

# 使用：
1. 无开机启动选项
2. 左键将当前图片设置为桌面壁纸
3. 右键删除当前图片
4. F5刷新显示的图片
